# Mautic-Nest

Full-stack marketing automation platform rebuilt from Mautic's architecture.

**Stack:** NestJS 11 + Next.js 16 + PostgreSQL + Mastra AI Agents

## Quick Start

```bash
# Install dependencies
pnpm install

# Set up database
cp .env.example .env.local
drizzle-kit push

# Run dev
pnpm dev
```
